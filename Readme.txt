

源码部署
    部署教程：
        1、	上传安装包源码
        2、	将安装包根目录下的data.sql文件导入到数据库中
        3、	修改根目录下的.env文件的信息，如需调请将APP_DEBUG=false的布尔值改为true
        4、	为保持稳定运行还需改动config/database.php文件中的数据库配置
        5、	将网站设定为伪静态根据运行环境设置相对于的规则
        6、	将网站的运行目录设定为public目录
        7、	完善配置config/web.php文件
        8、	进入后台修改默认账户信息[后台地址下文见]
        9、	* 建议到云端注册账号查看新消息
    
    
    目录介绍：
        1、	后台目录/admin
        2、	代理目录/agent
        3、	接口目录/api
        4、	前台目录/index
