易验云网络验证  V1.7s
===============

## 环境需求
> 运行环境要求PHP8.0+，兼容PHP8.1

> 配置伪静态规则以及设置运行目录为[public]


## 主要新特性
* 采用Think PHP6 开发 且兼容所有Think PHP所支持的

## 安装

~~~
git close https://github.com/yiyanyun/major.git
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~
